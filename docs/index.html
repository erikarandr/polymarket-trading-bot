Quick start

git clone https://github.com/erikarandr/polymarket-trading-bot.git
cd polymarket-trading-bot
npm install
cp .env.example .env
npm start

Requires Node.js ≥ 20.6. Set POLYMARKET_PRIVATE_KEY in .env.
